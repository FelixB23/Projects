Chapitre 5 : La fonction logarithme népérien

Définition

Soit y un réel strictement positif. On appelle logarithme népérien du réel y l'unique solution à l'équation \(e^x = y\). On note cette solution \(\ln y\).

La fonction logarithme népérien, noté \(\ln\), est la fonction définie sur \(\mathbb{R}_{+}^{*}\) par \(x \mapsto \ln x\).

On a :

  • \(\forall x \gt 0,~ e^{\ln x}=x\)
  • \(\forall x \in \mathbb{R},~ \ln (e^x)=x\)
  • \(\forall a,b \gt 0,~ \ln a = \ln b \Leftrightarrow a = b\) et \(\ln a \lt \ln b \Leftrightarrow a \lt b\)

La fonction \(\ln\) est la fonction réciproque de la fonction exponentielle.

Etude de la fonction logarithme népérien

La fonction \(\ln\) est définie, continue, et strictement croissante sur \(]0 ; +\infty[\). On a les limites suivantes :

  • \(\lim_{n \to 0} \ln x = -\infty\)
  • \(\lim_{n \to +\infty} \ln x = +\infty\)
  • \(\lim_{n \to 0} {\displaystyle \frac{\ln (1+x)}{x}} = 1\)
  • \(\lim_{n \to +\infty} {\displaystyle \frac{\ln (x)}{x}} = 0\)
  • \(\lim_{n \to 0} x\ln x = 0\)


La fonction \(\ln\) dérivable sur \(\mathbb{R}_{+}^{*}\). Sa dérivé est \((\ln x)' = {\displaystyle \frac{1}{x}}\).

Si \(u\) est une fonction dérivable et strictement positive sur un intervalle I, \(\ln u\) est dérivable sur I et \((\ln u)' = {\displaystyle \frac{u'}{u}}\).

Signe de la fonction \(\ln\) :

  • \(\ln x = 0 \Leftrightarrow x = 1 \)
  • \(\ln x \lt 0 \Leftrightarrow 0 \lt x \lt 1 \)
  • \(\ln x \gt 0 \Leftrightarrow x \gt 1 \)

Propriétés algébriques de la fonction logarithme népérien

Relation fonctionnelle : \(\forall a,b \gt 0,~ \ln(a \times b) = \ln a + \ln b\)

Propriété immédiates

  1. \(\ln {\displaystyle \frac{1}{b}} = -\ln b\)
  2. \(\ln {\displaystyle \frac{a}{b}} = \ln a - \ln b\)
  3. \(\ln a^n = n \times \ln a\)
  4. \(\ln \sqrt{a} = {\displaystyle \frac{1}{2}}\ln b\)
